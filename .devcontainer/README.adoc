= devcontainer

== Building S-CORE communication with bazel

.Build S-CORE communication
[source,shell]
----
git clone git@github.com:eclipse-score/communication.git
cd communication
bazel build ...
----

== IDE support (compile_commands.json)

https://github.com/kiron1/bazel-compile-commands[bazel-compile-commands] creates a `compile_commands.json` file which can then be used by IDEs / clangd for code completion and navigation.
The container has this tool preinstalled.

To create a `compile_commands.json` file, run the following command in the `communication` directory:

.Build compile_commands.json
[source,shell]
----
bazel build ...
bazel-compile-commands ...
----
